# Requirements

This folder contains functional and non-functional requirements for the LifeCostTracker project.

## Files
- `functional-requirements.md`: Functional requirements (features list)
- `non-functional-requirements.md`: Non-functional requirements (performance, security, UX)

## Summary
### Core Features
- **Daily Cost Perspective**: Every expense is broken down into daily cost
- **Expense Tracking**: Manual entry, categorization, receipt photos
- **Credit Life Management**: Credit cards, loans, installments
- **Subscription Management**: Subscriptions, free trials, reminders
- **Wishlist Planning**: Affordability checks, scenario planning
- **Dashboard & Reports**: Daily cost average, trends, reports

### Non-Functional Requirements
- **Performance**: App launch < 2s, dashboard load < 1s, daily cost calculation < 100ms
- **Security**: AES-256 encryption, TLS 1.3, biometric auth
- **Usability**: Onboarding < 3 minutes, accessibility support
- **Reliability**: Crash rate < 0.1%, offline support
- **Compatibility**: iOS 16.0+, Android 6.0+

# 需求文档

本文件夹包含 LifeCostTracker 项目的功能需求和非功能需求。

## 文件
- `functional-requirements.md`: 功能需求（功能列表）
- `non-functional-requirements.md`: 非功能需求（性能、安全、用户体验）

## 总结
### 核心功能
- **每日成本视角**: 每笔支出都分解为每日成本
- **支出追踪**: 手动录入、分类、收据照片
- **信用生活管理**: 信用卡、贷款、分期付款
- **订阅管理**: 订阅、免费试用、提醒
- **愿望清单规划**: 可负担性检查、场景规划
- **仪表盘与报告**: 每日成本平均值、趋势、报告

### 非功能需求
- **性能**: 应用启动 < 2秒，仪表盘加载 < 1秒，每日成本计算 < 100ms
- **安全**: AES-256 加密、TLS 1.3、生物识别认证
- **可用性**: 首次使用 < 3分钟、无障碍支持
- **可靠性**: 崩溃率 < 0.1%、离线支持
- **兼容性**: iOS 16.0+、Android 6.0+
