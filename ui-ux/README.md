# UI/UX Design

This folder contains UI/UX design guidelines for the LifeCostTracker project.

## Files
- `ui-ux-design.md`: UI/UX design guidelines (wireframes, user flow, visual design)

## Summary
### Design Principles
- **Daily Cost First**: Every expense is presented as a daily cost first
- **Clarity First**: Prioritize readability and simplicity
- **Gentle Onboarding**: Start simple, add complexity as user grows
- **Positive Reinforcement**: Celebrate small wins
- **Accessibility**: Design for all users

### Key Screens
1. **Home Dashboard**: 
   - Hero Section: 平均每日成本, 月度订阅支出, 全部項目
   - Sorting Toggle: 按性价比 / 按日耗 / 按分类
   - All Items List: 神仙性价比 section, item cards
2. **Add New Item Sheet**: Ownership question, usage days, real-time preview
3. **Item Detail View**: Hero section, history & stats, actions

### Color Palette
- Primary Blue: #007AFF
- Success Green: #34C759
- Warning Orange: #FF9500
- Danger Red: #FF3B30
- 神仙性价比 Gradient: From #34C759 to #007AFF

# UI/UX 设计

本文件夹包含 LifeCostTracker 项目的 UI/UX 设计指南。

## 文件
- `ui-ux-design.md`: UI/UX 设计指南（线框图、用户流程、视觉设计）

## 总结
### 设计原则
- **每日成本优先**: 每笔支出都先以每日成本展示
- **清晰优先**: 优先考虑可读性和简洁性
- **温和引导**: 开始简单，随用户成长增加复杂性
- **正面强化**: 庆祝小胜利
- **无障碍设计**: 为所有用户设计

### 关键页面
1. **首页仪表盘**: 
   - Hero 区域: 平均每日成本、月度订阅支出、全部項目
   - 排序切换: 按性价比 / 按日耗 / 按分类
   - 所有物品列表: 神仙性价比区域、物品卡片
2. **添加新物品表单**: 所有权问题、使用天数、实时预览
3. **物品详情视图**: Hero 区域、历史与统计、操作

### 调色板
- 主蓝色: #007AFF

- 警告橙色: #FF9500
- 危险红色: #FF3B30
- 神仙性价比渐变: 从 #34C759 到 #007AFF
